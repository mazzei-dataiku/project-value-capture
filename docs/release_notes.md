# Release Notes

## Version 1.1.2

Top features:

- **Customizable project intake macro** with configurable dropdowns and validation
- **Value Drivers capture** to standardize expected impact/value across projects
- **Hub logging (bronze dataset)** for centralized reporting and governance
- **Optional Snowflake variables provisioning** into the created project’s global variables
- **Snowflake user profile helpers** to load/save commonly used variable values
